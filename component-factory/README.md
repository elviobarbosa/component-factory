# Component Factory

Extensão para o Visual Studio Code que gera automaticamente novos componentes Angular
a partir de um modelo existente.  
Permite duplicar pastas de componentes, atualizando nomes, seletores e imports.

## Como usar

1. Clique com o botão direito em uma pasta de componente existente.
2. Selecione **"Gerar novo componente baseado em modelo"**.
3. Informe o novo nome (ex: `user-profile`).

Um novo componente será criado com todos os arquivos e nomes substituídos.
